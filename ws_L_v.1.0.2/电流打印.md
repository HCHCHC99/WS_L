# 电流打印设置 (JScope + VOFA+)

---

## 1. 硬件配置

| 项目 | 值 |
|------|-----|
| 电流传感器 | Vout = VCC/2 + 0.528 × Ip(A) (V) |
| 量程 | ±2.5A |
| 零点电压 | 1650 mV |
| 零点 ADC | 2048 raw |
| 灵敏度 | 528 mV/A, 655 counts/A, 1.53 mA/count |
| ADC 分辨率 | 12-bit, 3.3V ref |
| RC 滤波 | 100Ω + 1nF (fc ≈ 1.6 MHz) |
| PWM 频率 | **100 kHz** (中心对齐, TMR4_3) |
| 三相引脚 | PA5=IU(ADC1_CH5), PA6=IV(ADC1_CH6), PA7=IW(ADC1_CH7) |
| 采样触发 | TMR4_3 SCMP0 (PWM 峰值, 共享 BEMF 触发源) |
| 采样频率 | 50 kHz (每 PWM 周期采样一次) |
| EMA 滤波 | α = 8/256 ≈ 3.1%, fc ≈ 250 Hz |

---

## 2. 数据链路

```
ADC1 SEQ_B ISR (50kHz, PWM 峰值触发)
    │
    ├── raw ADC (u16) → I_ADC_TO_MA_REF() → 瞬时电流 (int16_t mA)
    │       │
    │       ├── g_i_iu_ma / g_i_iv_ma / g_i_iw_ma   (瞬时, 有 PWM 纹波)
    │       │
    │       └── EMA 滤波器 (Q8)
    │               │
    │               ├── g_i_iu_filt / g_i_iv_filt / g_i_iw_filt  (滤波后 ×256)
    │               │       │
    │               │       └── VOFA+: filt >> 8 → int32 mA → SendScaled(×0.001) → A
    │               │
    │               └── g_i_iu_disp = (filt>>8) + 10000  (uint16_t, 恒正)
    │                       │
    │                       └── JScope: 直接读取
    │
    └── g_i_uvw_ma = IU+IV+IW (应≈0)
```

---

## 3. JScope 设置

### 变量

| 变量 | 类型 | 含义 | 换算 |
|------|------|------|------|
| `g_i_iu_disp` | uint16_t | U 相滤波电流 + 10000 | 实际 mA = 值 − 10000 |
| `g_i_iv_disp` | uint16_t | V 相滤波电流 + 10000 | 实际 mA = 值 − 10000 |
| `g_i_iw_disp` | uint16_t | W 相滤波电流 + 10000 | 实际 mA = 值 − 10000 |
| `g_i_iu_filt` | int32_t | U 相 Q8 滤波电流 | 实际 mA = 值 / 256 |
| `g_i_iu_ma` | int16_t | U 相瞬时电流 | 实际 mA = 值 (有纹波) |
| `g_i_uvw_ma` | int32_t | 三相电流和 | 应 ≈ 0 |

### 配置方式

1. JScope 选 **HSS** 模式（高速采样）
2. 加载 ELF 文件，选择目标变量
3. 对于 `g_i_iu_disp`：表达式设为 `(g_i_iu_disp - 10000)` 即可显示 mA

---

## 4. VOFA+ 设置

### 发送配置

| 项目 | 值 |
|------|-----|
| 协议 | JustFloat |
| 通道 | CH1=IU, CH2=IV, CH3=IW |
| 发送周期 | **20 ms (50 Hz)** |
| 数据格式 | EMA 滤波后 int32 mA，×0.001f → float A |
| 帧大小 | 16 字节 (3×4 浮点 + 4 尾帧) |
| 发送方式 | DMA2 CH0 |
| 波特率 | 115200 |
| 带宽占用 | 50Hz × 16B = 800 B/s (7%) |

### 代码路径

```c
// main.c 主循环, 每 20ms:
int32_t cur[3];
cur[0] = (int32_t)(g_i_iu_filt >> 8);   // Q8 → mA
cur[1] = (int32_t)(g_i_iv_filt >> 8);
cur[2] = (int32_t)(g_i_iw_filt >> 8);
Usart3_Vofa_SendScaled(cur, 3, 0.001f);  // mA → A
```

### VOFA+ 上位机操作

1. 协议: **JustFloat**
2. 串口: **115200-8-N-1**
3. 左上角展开 JustFloat → 拖 **CH1/CH2/CH3** 到波形图
4. Y 轴单位: **A** (安培)
5. 参考: 1.234 A → 实际电流 1234 mA

## 5. 对比

| | JScope | VOFA+ |
|------|------|------|
| 数据源 | `g_i_iu_disp` (+10000 偏移) | `g_i_iu_filt >> 8` (EMA 滤波 mA) |
| 更新率 | ~kHz (HSS 轮询) | 50 Hz |
| 传输方式 | JTAG/SWD 内存读取 | USART3 DMA |
| 优点 | 速率高、不占 CPU | 无线缆、可远程 |
| 适用场景 | 调试、纹波分析 | 运行监控、长时间记录 |
